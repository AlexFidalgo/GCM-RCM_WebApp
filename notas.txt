After changes to the code
    docker compose up -d --build
    What it does:
        (A) Rebuilds images only if needed
            Docker checks whether files in the build context have changed.
            If yes: it rebuilds the image.
            If no: it skips the rebuild (fast).
        (B) Starts containers
            If containers already exist, Docker recreates them with the newly built image.
            If they donâ€™t exist, it creates new ones.
